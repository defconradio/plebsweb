# Plebs Web

Simple  Vite + React scaffold prepared for Vercel deployment.

Getting started

Install dependencies and run locally:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

Deploy to Vercel

Connect this repository in Vercel. The `vercel.json` uses `@vercel/static-build` and expects the `dist` directory produced by `vite build`.
